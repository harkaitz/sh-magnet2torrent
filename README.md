# SH-MAGNET2TORRENT - Convert magnet link to torrent file

## Help

magnet2torrent

    Usage: magnet2torrent URLS...
    
    Convert magnet URLs to torrent files using aria2c(1). The
    resulting filenames are printed to standard output.
